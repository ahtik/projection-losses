Projection losses
==================

Python implementation of "Structured Prediction with Projection Oracles".

Supported polytopes
-------------------

* Probability simplex
* Unit cube
* Knapsack polytope
* Birkhoff polytope
* Permutahedron
* Order simplex
* Cardinal products

Installation
------------

Simply copy relevant files to your project.
